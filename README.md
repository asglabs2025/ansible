# Ansible Configuration for Homelab VMs

This folder contains Ansible playbooks and inventory to configure VMs deployed via Terraform in this repo.

## Folder Structure

``` 
ansible/
├── inventory/ # Hosts file listing VMs
├── logs / # Log file folder
├── playbooks/ # Top-level playbooks
├── roles/ # Optional reusable roles
├── scripts/ # Scripts for crontab automation
└── ansible.cfg # Ansible configuration
```
